This repository contains a collection of basic C programs created for practice and learning fundamental programming concepts in the C language.
Each program focuses on a specific problem statement commonly taught in introductory C programming courses.

🛠️ Tools Used:<br>
Language: C<br>
Compiler: GCC (MinGW / MinGW-w64)<br>
Editor: Visual Studio Code<br>
Terminal: Git Bash<br>
OS: Windows<br><br>

▶️ How to Compile and Run:<br>
Compile a program:<br>
gcc filename.c -o filename
Run the program:<br>
./filename
Example:<br
gcc factorial.c -o factorial<br>
./factorial<br><br>

📌 Purpose of This Repository:<br>
Practice core C programming concepts<br>
Strengthen logic building<br>
Reference for beginners learning C<br>
