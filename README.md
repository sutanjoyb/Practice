This repository contains a collection of basic C programs created for practice and learning fundamental programming concepts in the C language.
Each program focuses on a specific problem statement commonly taught in introductory C programming courses.

🛠️ Tools Used:
Language: C
Compiler: GCC (MinGW / MinGW-w64)
Editor: Visual Studio Code
Terminal: Git Bash
OS: Windows

▶️ How to Compile and Run:
Compile a program:
gcc filename.c -o filename
Run the program:
./filename
Example:
gcc factorial.c -o factorial
./factorial

📌 Purpose of This Repository:
Practice core C programming concepts
Strengthen logic building
Reference for beginners learning C
