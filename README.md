📘 C Programming Fundamentals Repository

This repository contains a curated collection of fundamental C programs intended to support learners in understanding core programming concepts. The programs focus on essential topics including arrays, strings, searching algorithms, sorting techniques, and basic control structures. All examples are implemented using standard C syntax and are suitable for beginners, students, and those revising foundational concepts. The repository serves as a structured reference and practical learning resource for academic coursework and self-study.

---

▶ How to Run the Programs (Using Git Bash)

1. Open Git Bash on your system.<br>

Navigate to the repository directory:<br>

cd path/to/repository<br><br>


2. Compile the C source file using GCC:<br>

gcc filename.c -o filename<br><br>


3. Execute the compiled program:<br>

./filename<br><br>


View the output displayed in the terminal.<br>

**Note:** <br>

Replace filename with the actual C file name.<br>

Ensure GCC is installed and properly configured before compiling.
